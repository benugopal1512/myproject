# My PHP Project

## 📌 Objective

Set up a local development environment using XAMPP and create a basic PHP project.

---

## 🛠️ Tech Used

* PHP
* Apache (XAMPP)
* MySQL (XAMPP)
* Git & GitHub

---

## ⚙️ Setup Instructions

1. Install XAMPP
2. Start Apache and MySQL from XAMPP Control Panel
3. Create project folder in:
   C:\xampp\htdocs\myproject
4. Create `index.php` file
5. Run in browser:
   http://localhost:8080/myproject
